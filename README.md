# 통합정보시스템

Intergrated_Informatino_System 프로젝트는 학교 통합정보 시스템을 입니다. 크게 3가지 역할이 있으며 다음과 같습니다.

## 특징

### 학생

- 교과관리(교과목 확인, 강의시간표 확인)
- 학적관리(학적기본, 학적변동)
- 장학관리(장학현황)
- 수강관리(수강신청, 수강취소, 강의계획서 확인)
- 성적 관리(당학기 성적조회, 전체 성적 조회, 졸업 자가진단)

### 교수

- 수강관리(강의계획서 등록)
- 성적관리(성적처리)

### 관리자

- 교과관리(교과목 등록, 교과목 삭제)
- 학적관리(학번부여, 학적변동)
- 장학관리(장학등록)
